About Gainly

Gainly is a fitness and nutrition tracker designed to make daily progress simple: log meals, track calories and macros, record workouts, and stay consistent. It is built to reflect a real SaaS-style product with authentication, a dashboard, and premium feature previews.

Track Meals

Add meals with calorie and macro information and see your totals for the day. Gainly calculates your remaining calories and visualizes your daily progress.

Log Workouts

Record workouts with exercises including sets, reps, and weight. Gainly calculates total volume and provides a clear summary of daily activity.

Monitor Progress

The dashboard displays key daily statistics and recent activity. Premium features are previewed and can be unlocked for demonstration purposes.

Technology Stack

  • Frontend: HTML, CSS, Vanilla JavaScript
  • Backend: Node.js with Express
  • Authentication: JWT with bcrypt password hashing
  • Deployment: Render
  • Storage: Local storage (frontend demo), in-memory server storage

Portfolio Project

Gainly is built as a portfolio-ready full-stack application demonstrating authentication, protected routes, interactive UI components, and structured client-side state management.
